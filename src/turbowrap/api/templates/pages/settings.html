Come ottenere un token GitHub:

Linear Integration

Per gestire issue e sincronizzazione

Configurato Non configurato

Come ottenere API key e Team ID:

  1. Vai su Linear Settings > API > Personal API keys
  2. Crea una nuova API key con permessi read e write
  3. Copia la API key (lin_api_...)
  4. Trova il Team ID nell'URL della tua team (ultima parte UUID)
